# SWB — Typografie

De lettertypes van het merk SWB.

## Basislettertype — Basic Sans

Voor alle reguliere communicatie. Beschikbare gewichten:

- Basic Sans Light (+ italic)
- Basic Sans Regular (+ italic)
- Basic Sans SemiBold (+ italic)
- Basic Sans Bold (+ italic)
- Basic Sans Black (+ italic)

## Logo-lettertype — Aptly

Alleen te gebruiken bij een **nieuw logo** of **in overleg**. Niet voor gewone teksten.

## Pay-off — Pacifico Bold

**Uitsluitend** voor de pay-off. Dit font verder **niet** gebruiken.
