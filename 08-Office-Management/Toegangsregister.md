# Toegangsregister

Overzicht van wie welke fysieke toegang heeft (sleutel/pas). Bijwerken bij **elke on- en
offboarding**.

> 🔒 **Beveiliging — belangrijk.** Bewaar **géén alarmcodes, wachtwoorden of pincodes** in
> dit document. Dit brain kan een AI-chatbot voeden; secrets horen daar nooit in. Noteer
> alleen *of* iemand een alarmcode kent (ja/nee), niet de code zelf. Bewaar de echte codes
> in een beveiligde wachtwoordkluis.
>
> ⚠️ Het bronbestand bevatte alleen kolomkoppen (nog geen gegevens). Vul hieronder de
> daadwerkelijke uitgiftes in.

| Naam | Functie / afdeling | Sleutel / pas | Alarmcode bekend? (ja/nee) | Sinds | Ingenomen op (bij vertrek) | Opmerking |
|------|--------------------|---------------|----------------------------|-------|----------------------------|-----------|
| | | | | | | |

## Werkwijze

- **Onboarding:** voeg een rij toe zodra iemand een sleutel/pas krijgt.
- **Offboarding:** vul "ingenomen op" in en wijzig de alarmcode als die persoon 'm kende
  (zie [[Proces-Onboarding-en-Offboarding]]).
